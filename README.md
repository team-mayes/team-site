# Website for Team Mayes & Blue

# Usage
Be sure to have both the `jekyll` and `jekyll-paginate` gem installed.

Then using your favorite terminal do:

```
$ git clone git@github.com:team-mayes/team-site.git
$ cd team-site
$ jekyll serve
```
